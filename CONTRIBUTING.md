# Contributing

Thanks for helping improve Awesome AgentOps.

This repository is meant to be a focused engineering resource for operating AI agents in production. Please keep additions practical, specific, and relevant to AgentOps.

## What Belongs Here

Good additions usually help teams with one or more of these areas:

- Observability, tracing, monitoring, or telemetry for agents.
- Evaluation, testing, regression suites, or benchmarks.
- Replay, debugging, run inspection, or failure analysis.
- Guardrails, runtime policy checks, approval flows, or control planes.
- Security, identity, access control, sandboxing, or red teaming.
- Deployment, reliability, cost, latency, or incident response.
- Governance and auditability for agentic systems.
- Multi-agent orchestration with a production operations angle.

Avoid adding:

- Generic AI product directories.
- Prompt packs without testing or evaluation relevance.
- Thin wrappers around a model API with no operational value.
- Unmaintained projects unless they are historically important.
- Marketing pages that do not provide useful technical substance.

## Pull Request Guidelines

Before opening a pull request:

1. Check that the resource is not already listed.
2. Add it to the most specific section.
3. Use the format `- [Name](https://example.com/) - Short description.`
4. Keep the description factual and concise.
5. Explain why the resource belongs in AgentOps if the connection is not obvious.

Descriptions should say what the resource helps operators do. Avoid hype words like "revolutionary", "best", "cutting-edge", or "game-changing".

## Quality Bar

Entries are more likely to be accepted when they have:

- Public documentation or source code.
- Clear relevance to production agent operations.
- Evidence of maintenance, adoption, or durable reference value.
- A concrete operational use case.

Commercial tools are allowed if they are relevant and useful, but the list should not become a vendor directory.

## Issues

Issues are welcome for:

- Suggesting resources.
- Reporting dead links.
- Proposing new categories.
- Challenging whether a category or entry belongs here.

Please include a link and a short explanation of the production AgentOps problem the resource addresses.

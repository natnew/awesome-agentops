# Security Policy

This repository is an Awesome list and does not currently ship executable software.

## Reporting a Security Concern

If you find a security concern in this repository, such as a malicious link, suspicious resource, or unsafe recommendation, please open a private security advisory on GitHub if available.

If private advisories are not available, open an issue with enough detail to identify the concern. Do not include exploit instructions or sensitive information in public issues.

## Resource Safety

Resources linked from this repository are maintained by their respective authors. Inclusion does not mean endorsement, warranty, or a security review.

When evaluating AgentOps tools for production use, review:

- Data handling and retention.
- Secret and credential management.
- Access control and audit logs.
- Network egress and third-party integrations.
- Model, prompt, trace, and user-data storage.
- Compliance requirements for your environment.

## Supported Versions

The `main` branch is the only supported version of this list.
